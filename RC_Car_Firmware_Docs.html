The default startup mode. Uses ultrasonic distance and a servo sweep to scan left and right before choosing a turn direction.

Obstacle Avoidance States

The active obstacle logic behaves like a small state machine: move, scan, turn, and return to moving.

State machine
1

MOVE

Update speed and check front distance. If clear, keep speed high. If close enough, go to SEARCH.

2

SEARCH LEFT

Servo moves toward 170 degrees and stores leftDist.

3

SEARCH RIGHT

Servo moves toward 10 degrees and stores rightDist.

4

TURN

Choose a diagonal turn, recenter the servo, and return to MOVE.

Emergency stop-distance check

  • If the state is not TURN, the code measures stopDist.
  • If stopDist <= STOP_DIST, it immediately calls backward().
  • It sets currentSpeed = 0 and returns.
  • This safety branch runs before regular scan logic.

Key constants

  • BREAK_DIST = 50
  • STOP_DIST = 10
  • HIGHEST_SPEED = 180
  • LOW_SPEED = 20
  • CENTER_ANGLE = 90
  • LEFT_SCAN_ANGLE = 170
  • RIGHT_SCAN_ANGLE = 10

Bluetooth Commands

The current sketch reads a single Bluetooth character at a time. That is why the live code reacts to single digit speed input instead of long packet strings.

Input logic

Mode selection

N = line follower, O = obstacle mode, M = manual mode, W = follow-me entry.

Power and Reset Notes

The logs showed that resets were caused by hardware power load around the servo and motors, not by normal mode-switch code.

Critical lesson

What the logs proved

  • In logs.txt, [SYS] ready appeared multiple times during runtime.
  • That means the board restarted during operation.
  • In logs2.text, the startup banner appeared only once.
  • So the reset source was power drop or noise, not normal logic flow.

Recommended wiring practice

  • Do not power the servo from the Arduino 5V pin during full operation.
  • Do not power motors from the Arduino board.
  • Use an external motor supply through the motor driver.
  • Use a stable 5V source for the servo.
  • Keep a common ground across Arduino, driver, servo, Bluetooth, and sensors.
  • Add a 470uF to 1000uF capacitor near the servo supply.
